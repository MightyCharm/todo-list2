Project: ToDo List (The Odin Project: JavaScript Course)

## Overview ‚ÑπÔ∏è

In this program, you can create projects and todos inside that specific project. If you want to change
things up, you can edit the project names or everything inside your todos. If a project or todo is not needed
anymore, it can be deleted,
Everything will be saved inside localStorage, so data won't be lost when the program restarts.

- npm and webpack
- localStorage saves and reloads the data
- mobile first approach
- create, edit, delete projects
- create, edit, delete todos
- kebab buttons for edit/delete projects/todos
- input validation(simple but can be built upon) with visual feedback
- todos can be expanded to show more information
- visual feedback if a todo is done
